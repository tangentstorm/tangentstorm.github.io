1 the core loop

The following general procedure describes the behavior of an arbitrary interactive program in the syntax of the programming language pascal. 

procedure coreloop;
  begin
    with context do
      repeat
        observe; evaluate; respond
      until done
  end;

2 the current context

The with … do … construct tells the compiler that the identifiers in the following statement needn't be defined globally, but may instead refer to components of the specified object.

In this case, the object is specified by the word context, but that identifier could be defined in a number of different ways: as a constant value, as a variable bound to an object, as a function that produces an object, or as a variable bound to a function that produces an object.

As it happens, we will take this last option, as it is the most flexible, and define context like so: 

var context : TCtxFn;

The word var instructs the compiler to process a series of variable declarations. The above line defines a single variable, context, which may refer to any value of the type TCtxFn.
