1 Introduction

Implish is a small imperative programming language that blends elements of forth, pascal, and APL with a notation inspired by smalltalk and lisp.

2 Motivation

I wanted to make a language that was extensible and easy to bootstrap, combining the best ideas of lisp, forth, and smalltalk.

  • minimal syntax
  • support for quoting and templating
  • blend of functional, logical, and imperative control flows

3.1.3 discussion

By default, items are simply placed on a stack, and (as in forth) verbs take their arguments immediately.

Parentheses allow execution of the first token to be deferred until the entire list is parsed.

4 Lexical Analysis

COMMENT
  : '#' (Any)*  '\n' { $channel = HIDDEN; }
  ;

STRING
  : '"' ( Esc | ~'"' )* '"'
  ;

KEYWORD
  : Alpha ( Alp ) ':'
  ;

fragment Space = #0 .. #32 ;
fragment Alpha = 'A'..'Z' | 'a'..'z' ;
fragment Digit = '0'..'9' ;
fragment Hexit = Digit | 'A' .. 'F' | 'a' .. 'f' ;
fragment Esc = '\\' ( 'n' | 't' | '\\' | '"' ) ;
fragment Any = #0 .. #255 ;

5 Notation

Implish has a simple, flexible syntax. 

program
  : ( expr )*
  ;

expr // a single expression
  : block | quote | phrase | number | STRING | WORD | SYMBOL
  ;

block
  : '[' program ']'  // quotation (also list, tuple, etc)
  | '(' program ')'  // prefix block (lisp-like expressions)
  | '{' program '}'  // lambda
  | ':' iden '|' program ';'  // function definitons
  ;

quote
  : "'" expr // normal quote
  | '`' expr // quasiquote
  | ',' expr // unquote
  | '@' expr // splice
  ;

number
  : Digit +    // decimal
  | '$' Hexit+ // hexidecimal
  ;

phrase
  : (KEYWORD expression)+
  | expr OPERATOR expr
  ;
