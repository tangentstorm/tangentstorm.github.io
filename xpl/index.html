1 narrative code

These units are written with a literate programming style:

arrays
generic array components
bplus
an implementation of B+ trees.
kvm
a modern crt, providing a 256 color, unicode-enabled text terminal
sets
generic sets of abitrary type (sets of records, classes, etc)
